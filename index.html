Sobre Mim

Sou apaixonado por tecnologia desde jovem e essa paixão me levou a cursar Análise e Desenvolvimento de Sistemas. Durante minha formação e carreira, desenvolvi diversas habilidades práticas que me permitem criar soluções web completas e funcionais.

Tenho experiência sólida no desenvolvimento de sites e landing pages, trabalhando com HTML, CSS e JavaScript para criar interfaces modernas e responsivas. Utilizo frameworks como Bootstrap e jQuery para agilizar o desenvolvimento, sempre focando na experiência do usuário.

No backend, trabalho com PHP e Java, criando APIs robustas e sistemas escaláveis. Tenho conhecimento em SQL para modelagem e manipulação de bancos de dados, além de experiência com controle de versão usando Git.

Atualmente, estou expandindo minhas habilidades para o desenvolvimento mobile, buscando criar aplicações multiplataforma que proporcionem experiências excepcionais em dispositivos móveis. Também utilizo UML para documentação e planejamento de sistemas.

Estou sempre em busca de novos desafios e oportunidades para aplicar e expandir meus conhecimentos técnicos.
